# Art Source

This directory contains editable or high-resolution source art required for future asset production.

- `evolutions/` contains three-stage monster source sheets.
- `monster-packs/` contains monster action source sheets and batch references.
- `layers/` contains environment and cannon source images.
- `legacy/` contains retired assets that may still be useful as references.

PNG files in this directory are tracked with Git LFS. Runtime-ready output belongs under `assets/`. Previews, diagnostics, backups, and temporary exports belong under `local-output/`.

Do not reference files in this directory from the game runtime.
