# BGM Asset Drop Folder

Put exported BGM files in this folder.

Preferred format:
- MP3, 128-192 kbps, stereo
- 60-90 seconds
- Seamless loop
- No vocals, no lyrics, no fade out

Optional later:
- OGG versions can be added with the same base names if needed.

## Required First Batch

| File name | Use |
| --- | --- |
| `bgm-01-title-theme.mp3` | Home screen, mode selection |
| `bgm-02-study-manual.mp3` | Learning handbook, review, mastery, monster dex |
| `bgm-03-sunny-grassland.mp3` | Route 01 / Sunny Grassland battle |
| `bgm-04-candy-forest.mp3` | Route 02 / Candy Forest battle |
| `bgm-05-crystal-ice-cave.mp3` | Route 03 / Crystal Ice Cave battle |
| `bgm-06-mechanical-city.mp3` | Route 04 / Mechanical City battle |
| `bgm-07-astral-ruins.mp3` | Route 05 / Astral Ruins battle |
| `bgm-08-boss-battle.mp3` | Boss / ultimate-stage battle |

## Export Notes

- Keep the volume roughly consistent across all tracks.
- Avoid long intros. Gameplay should feel active within the first 2-4 seconds.
- If Suno creates a fade out, trim or regenerate because looping will feel broken.
- Keep file names exactly as listed above so the game can load them automatically later.
